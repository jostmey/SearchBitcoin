The files in the directory "api.searchbitcoin.com" ran on a LEMP server. The
server was setup using the following packages.
 * OS: Debain 6
 * Webserver: Nginx
 * Datbase: MySQL 5
 * PHP: 5

The website files were placed in the folder "/usr/share/nginx/www". Supporting
packages were placed one directory lower in "/usr/share/nginx".
